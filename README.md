## x-dev-uni

常用开发工具集和物料组件库,使用 pnpm monorepo 管理。

## 安装

```bash
# 确保 pnpm升级到 9.8.0 以上 npm i -g pnpm@latest
pnpm i

pnpm run build  # 开发子包使用pnpm run dev
```

## 文档

```bash
# cd /docs
pnpm run docs:dev
```
